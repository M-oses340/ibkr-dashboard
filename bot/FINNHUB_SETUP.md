# 🚀 Finnhub Setup - Fast Real-time Stock Data

## Get Your Free API Key (30 seconds)

1. Go to: https://finnhub.io/register
2. Sign up with email (free account)
3. Copy your API key
4. Paste it in the bot configuration

## Free Tier Limits
- ✅ 60 API calls per minute
- ✅ Real-time stock quotes
- ✅ No credit card required
- ✅ Unlimited for personal use

## Speed Comparison
- Finnhub: 0.2 seconds ⚡⚡⚡⚡⚡
- yfinance: 2-3 seconds 🐌🐌🐌

## After You Get Your Key

Replace `FINNHUB_API_KEY = "demo"` with your actual key in:
- `portfolio_bot.py`
- Or set environment variable: `export FINNHUB_API_KEY="your_key_here"`

That's it! The bot will be 10x faster!
